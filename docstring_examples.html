docstring_examples.google module¶

Example Google style docstrings.

This module demonstrates documentation as specified by the Google Python Style Guide. Docstrings may extend over multiple lines. Sections are created with a section header and a colon followed by a block of indented text.

Example

There are multiple ways to use Example or Examples, in this case we use a single colon : allowing us to use text.

If we wish to display a codeblock you may use double-colon ::

Just like this

Example:

my_string = "Alternatively you can just use a codeblock with double-colon"
string2 = "This is just an example that will remain unexecuted"

Examples:

>>> foo = "You can add chevrons to enable doctest on your codeblock"
>>> " ".join(foo.split(" ")[::-1])
codeblock your on doctest enable to chevrons addcan You

Section breaks are created by resuming unindented text. Section breaks are also implicitly created anytime a new section starts.

docstring_examples.google.example_of_keywords()[source]¶

Napoleon keyword.

There are a many keywords provided by napoleon which inform rendereing of the documentation. Each section will be rendered and highlighted differently.

Danger

When something really bad could happen!

Warning

When something could go wrong if you aren’t careful.

Note

When you want to bring something to attention, but it’s not super important.

Important

Make sure to read this, it contains key information.

Tip

Quick hints to help the struggling developer.

See also

Related subjects to help build context.

Example

How to use the thing. Include a codeblock:

print("Here is some code")

Or even a doctset:

>>> print("And now it's running")
And now it's running
